# hexo是什么
是一个基于node的静态博客引擎


------
# 安装

### 准备
- git
- node环境（包括npm）

### 安装
运行以下指令安装脚手架hexo-cli
```
npm install -g hexo-cli 
hexo -v # 验证是否安装成功
npm install # 安装依赖
```

------
# hexo常用指令

### 初始化
```
hexo init [folderName] 
```

### 生成静态网站
```
hexo g
``` 

### 启动本地服务
```
hexo s [-p port] # 默认端口4000（容易访问不到）,可以自己指定端口启动
```
